# PROJEK-1 — Aplikasi Komputer & Pemrograman (Excel)

Repositori ini berisi pengerjaan tugas Excel untuk mata pelajaran  
**Aplikasi Komputer & Pemrograman — PROJEK 1**.

Target pembelajaran:
- Mengolah data dengan fungsi **IF** (logika)
- Menggunakan **VLOOKUP** + Filtering / Sorting
- Membuat **Pivot Table dan Pivot Chart**
- Menyajikan insight sederhana dari data

> Tugas dibagi menjadi 3 folder: **TB1**, **TB2**, dan **TB3**.  
> Setiap folder memiliki file hasil + README penjelasan langkah pengerjaan.

---

## Tools yang digunakan
- Microsoft Excel (365/2021/2019/2016)
- Regional setting Indonesia → separator rumus `;` dan desimal `,`
- Untuk import data: Data → From Text/CSV

---

## Struktur Folder

| Folder | Isi | Output |
|--------|-----|--------|
| `TB1_Fungsi_IF` | Load CSV + rumus IF & nested IF | File `TB1.xlsx` |
| `TB2_VLOOKUP_Filter` | VLOOKUP ganjil/genap + Filtering employee | File `TB2.xlsx` |
| `TB3_Pivot_Chart` | Pivot Table, Pivot Chart, Insight | File `TB3.xlsx` |

---

## Cara menjalankan
Tidak perlu coding — cukup buka Excel dan ikuti langkah di README tiap folder:

1. Masuk ke folder tugas `TB1`, `TB2`, atau `TB3`
2. Baca README di dalamnya
3. Buka template Excel → ikuti langkah → export hasil
